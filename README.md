## API NestJs

### Os requisitos de sistema são:

1. O sistema armazena dados dos carros no sistema.
2. Cada cliente tem perfil cadastrado no sitema.
3. Cada cliente tem uma conta associada ao perfil.
4. O sistema tem dois tipode de perfils: Administrador e Cliente. Cada tipo dá acesso a recursos diferentes.
5. Administradores também têm perfil e conta cadastrados.
6. Administradores podem buscar, cadastrar, editar e deletar informações dos carros.
7. Administradores podem buscar, cadastrar, editar e deletar informações dos clientes.
8. CLientes podem buscar informações dos carros.
9. As buscas dos carros devem ser paginadas e aceitarem filtros de busca por nome e marca.

Tecnologias utilizadas:
- Framework NestJs
- Banco de dados relacional PostgreSQL
- Padronização REST
- Senhas Criptografadas
